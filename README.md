## Features

- User can **Login** & **Register**.
- User can **Add** , **Edit** & **Delete** Recipes owned by them.
- **Admin** has all above options for every Recipe uploaded by users.
- Responsive design



## Tech Stack Used in this Project

Angular , json-server, tailwind
